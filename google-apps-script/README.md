# EPIC Bikini Contest Model Card Automation

Live Google Sheet owner: `book@epicmodelsandtalent.com`

Live Sheet:
https://docs.google.com/spreadsheets/d/1zpRxGl9cBad2m5vJZmIM-gUaYUb-IB0aWYQPDbUmj6A/edit

Live Apps Script Web App:
https://script.google.com/macros/s/AKfycbxVwGX70-fL-QM1nfKqlSyNdrh0hq_CFwBsKvwYgZ_AEbJL6oLufGXzLLqP6zGEtlCN/exec

Tabs:
- Submissions
- Approved Models
- Votes
- Settings

Workflow:
1. User submits an entry.
2. Apps Script stores the entry in Submissions.
3. Applicant receives a confirmation email.
4. `book@epicmodelsandtalent.com` receives a review email with Approve / Reject links.
5. Approved entries move into Approved Models.
6. OG homepage and OG apply page read approved model cards from the Web App.
7. Visitor votes are saved to Votes and displayed in the fan tracker.

Public disclaimer:
Visitor voting is for fun and entertainment. Final contestant selection is made by EPIC Models & Talent staff, but fan votes may sway the judges.
